# Currency Converter - Конвертер валют

## Описание программы
Currency Converter - программа для конвертации валюты на актуальный курс.
### Возможности программы
- Выбор конвертируемой валюты
- Самостоятельное внесение суммы
- Сохрание истории конвертации

### Проверка корректности ввода:
- **Дата проведения конвертации:** должно отображаться актуальное время
- **Курс валюты:** должно переводить по актулаьному курсу валюты
