## Описание программы
Currency Converter - Простое GUI-приложение для конвертации валют с использованием актуальных курсов от ExchangeRate-API. История операций сохраняется локально в JSON.
### Возможности программы
- Выбор конвертируемой валюты
- Самостоятельное внесение суммы
- Сохрание истории конвертации

## Использование
- Выберите исходную и целевую валюты из выпадающих списков.
- Введите положительную сумму (например, 100).
- Нажмите «Конвертировать».
- Результат отобразится под кнопкой, а запись добавится в таблицу истории.
- При следующем запуске история загрузится автоматически.

## Как получить API-ключ
- Зайдите на [ExchangeRate-API](https://www.exchangerate-api.com/).
-  Нажмите **«Get Free API Key»**, зарегистрируйтесь с email.
-  В личном кабинете скопируйте ключ.
-  файле `converter.py` в переменной `API_KEY` замените `"ВАШ_API_КЛЮЧ"` на ваш ключ.

### Проверка корректности ввода:
- Тест 1: 100 USD → EUR. При курсе 0.92 результат ≈ 92.00 EUR.
- Тест 2: неправильный ввод: -5 или abc. Появится сообщение об ошибке.
- Тест 3: проверка истории: закройте и откройте программу – предыдущие конвертации останутся в таблице.
- Запуск без интернета: приложение выдаст ошибку загрузки курсов и закроется.
- Проверка предельных значений: очень большая сумма (1e10) или очень маленькая (0.0001).
- Проверка сохранения истории: закройте приложение, откройте history.json – файл должен содержать корректный JSON с записями.
